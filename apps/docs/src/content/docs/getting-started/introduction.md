# Minions Pipeline

Funnel stage tracking across the full job search lifecycle

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-pipeline/sdk`) — Core library for Node.js
- **Python SDK** (`minions-pipeline`) — Python bindings
- **CLI Tool** (`@minions-pipeline/cli`) — Command-line interface
- **Web App** — Interactive playground at [pipeline.minions.wtf](https://pipeline.minions.wtf)
