---
title: Installation
description: Install Minions Pipeline packages
---

## TypeScript / Node.js

```bash
npm install @minions-pipeline/sdk minions-sdk
```

## Python

```bash
pip install minions-pipeline
```

## CLI

```bash
npm install -g @minions-pipeline/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
