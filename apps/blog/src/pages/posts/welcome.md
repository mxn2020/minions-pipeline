# Welcome to Minions Pipeline

We're excited to introduce **Minions Pipeline** — Funnel stage tracking across the full job search lifecycle.

## What's Included

- **TypeScript SDK**: `npm install @minions-pipeline/sdk`
- **Python SDK**: `pip install minions-pipeline`
- **CLI**: `npm install -g @minions-pipeline/cli`
